# Installation

## Requirements

- Python 3.10 or later

## Install from PyPI

```bash
pip install confwire
```

Or, if you use [uv](https://docs.astral.sh/uv/):

```bash
uv add confwire
```

## Install from source

```bash
git clone https://github.com/sri-dhurkesh/confwire.git
cd confwire
uv sync
```
